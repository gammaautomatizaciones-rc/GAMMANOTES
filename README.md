# Anotador de Tareas

Un anotador sencillo de tareas pendientes, subdividido por categorías.

## Funcionalidades

- Crear categorías personalizadas.
- Agregar tareas dentro de cada categoría.
- Cambiar el estado de las tareas: Pendiente, En Progreso, Completada.
- Los datos se guardan localmente en el navegador usando localStorage.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Agrega una categoría escribiendo su nombre y haciendo clic en "Agregar Categoría".
3. Dentro de cada categoría, agrega tareas y cambia su estado con el selector.

## Subir a GitHub

1. Instala Git si no lo tienes.
2. Inicializa el repositorio: `git init`
3. Agrega los archivos: `git add .`
4. Haz commit: `git commit -m "Initial commit"`
5. Crea un repositorio en GitHub.
6. Agrega el remote: `git remote add origin <URL del repo>`
7. Sube: `git push -u origin main`

## Archivos

- `index.html`: Estructura de la página.
- `styles.css`: Estilos CSS.
- `script.js`: Lógica JavaScript.
